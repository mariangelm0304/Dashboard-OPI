📊 Importar datos desde Excel

Paso 1: Descargar plantilla

Descarga la plantilla de Excel con el formato correcto. Tiene 3 hojas: Calidad, Inventario y Pistoleo.

Paso 2: Llenar los datos en Excel

Abre el Excel, agrega o modifica los datos en las hojas correspondientes. Respeta los nombres de columnas.

Hoja "Calidad": columnas → Zona, Tienda, Tecnico, Semana, Estado, Fecha, Novedad
Hoja "Inventario": columnas → Zona, Tienda, Coordinador, Semana, Reporto, Fecha, Bodega, Cierre100, Fotos, Novedad
Hoja "Pistoleo": AG, Tienda, Zona, S1, S2, S3, ... S19 (o las semanas que tengas)
💡 Valores válidos:
Calidad - Estado: verde, amarillo, rojo, rojo-dark, pendiente
Inventario - Reporto: si, no, vacaciones, pendiente
Inventario - Cierre100: true, false, o vacío

Paso 3: Cargar el Excel

Selecciona el archivo Excel modificado para actualizar el dashboard.

Paso 4: Aplicar cambios

Revisa los datos importados y haz clic en "Guardar cambios" para actualizar el dashboard.

Cómo usar el editor

📋 Bitácoras de Calidad e Inventario Cíclico

  1. Selecciona la semana que quieres editar (1, 2, 3 o 4 de Mayo)
  2. Haz clic en "Cargar semana"
  3. Para cada tienda, selecciona el estado y completa los campos
  4. Haz clic en "Guardar cambios" al final
  5. Después haz clic en "Descargar archivo actualizado" y reemplaza el HTML actual

🎯 Estados de Calidad

  • Verde: Con reporte enviado a tiempo
  • Amarillo: Novedad/motivo (vacaciones, etc.)
  • Rojo oscuro: No envió reporte
  • Rojo: Sin técnico asignado (estructural)
  • Pendiente: Semana aún no ocurrida

📦 Estados de Inventario

  • Reportó + Cierre 100%: Envió reporte Y bodega cerrada 100%
  • Reportó (cierre parcial): Envió reporte pero cierre parcial
  • Vacaciones: Coordinador en vacaciones (no aplica)
  • Sin reporte: No envió información

🎯 Pistoleo

Cada semana que pasa, haz clic en "Agregar nueva semana" e ingresa el conteo de cada tienda. Deja vacío si no hay dato.

💾 Persistencia

Los cambios se guardan en el navegador (localStorage). Para compartirlos con tu equipo, descarga el HTML actualizado.
